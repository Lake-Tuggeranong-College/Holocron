# Report

In your report to your manager, you are required to explain and analyse the product you and/or your team is producing.

There are three sections to this report:

1) Project Milestones: A spreadsheet using the template provided in Google Sheets that covers the project milestones. The spreadsheet must be included and accompanied by an **analysis** and **critical evaluation** of the project milestones.  
2) Product Features : An **explanation** of the planned product features within the project. These **explanations** must be accompanied by some form of evidence, **analysis**, and **critical evaluation.**  
3) Data Flow: Using a diagram, **explain** and **analyse** how data is transmitted within different components of the project.
