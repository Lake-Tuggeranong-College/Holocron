# IP Address Subnetting

*   **What is Subnetting?**
    *   Dividing a large network into smaller, more manageable networks (subnets).
    *   Think of it like dividing a city into neighbourhoods – easier to manage!

# Why Subnet? – The Benefits
*   **Improved Security:** Isolates network segments, limiting the impact of security breaches.
*   **Enhanced Performance:** Reduces network congestion and improves data transmission speeds.
*   **Increased Efficiency:** Simplifies network administration and troubleshooting.

---

# Key Concept: Subnet Masks

*   Subnet masks are *crucial* for subnetting.
*   They define which part of the IP address represents the network and which represents the host.
